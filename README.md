# Fund_Raising_Dashboard
📊 Fundraising Analytics Dashboard | Power BI
📌 Project Overview

Developed an interactive Power BI dashboard to analyze fundraising performance across organizations, campaigns, fundraising platforms, and regions. The dashboard helps monitor KPIs, evaluate marketing ROI, analyze donor engagement, and support strategic decision-making through data-driven insights.

🎯 Project Objectives
Track fundraising KPIs and campaign performance.
Compare fundraising goals with actual funds raised.
Analyze donor engagement and marketing effectiveness.
Identify top-performing fundraising methods and platforms.
Support strategic planning using RCA and SWOT analysis.


📈 Dashboard 1: Overview Analysis
KPI Cards (Total Funds, Goal, Amount Raised, Marketing Spend, Social Media Reach)
Amount Raised by Organization
Fundraising Platform Analysis
Fund-wise Performance
Interactive Filters (Campaign Type & Fundraising Method)


📊 Dashboard 2: Methods, Strategies & Process Analysis
Profit Analysis by Fundraising Method (Waterfall Chart)
Campaign Type Distribution
Monthly Amount Raised vs Marketing Spend
Marketing ROI and fundraising trend analysis


🎯 Dashboard 3: Strategic Decision Making
Fishbone (Ishikawa) Diagram for Root Cause Analysis
SWOT Analysis (Strengths, Weaknesses, Opportunities & Threats)
Actionable recommendations for process optimization


💡 Key Insights
Tracked $7M raised against a $13M fundraising goal.
Compared fundraising performance across organizations and platforms.
Evaluated campaign effectiveness and marketing ROI.
Identified operational bottlenecks and improvement opportunities using RCA and SWOT.


🛠️ Tools & Technologies
Power BI Desktop
Power Query
DAX
Microsoft Excel
Data Modeling (Star Schema)
Interactive Visualizations & Slicers


🚀 Project Outcome

Built an interactive analytics solution that enables organizations to monitor fundraising performance, optimize marketing investments, improve donor engagement, and make data-driven strategic decisions.
