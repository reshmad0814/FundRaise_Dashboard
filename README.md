# 📊 Fundraising Analytics Dashboard | Power BI

An interactive **Power BI dashboard** designed to analyze fundraising performance across organizations, campaigns, fundraising platforms, and regions. This project provides actionable insights into fundraising efficiency, donor engagement, marketing ROI, and strategic decision-making.

---

## 📌 Project Overview

This dashboard helps organizations monitor fundraising performance by tracking key metrics, evaluating campaign effectiveness, and identifying opportunities for improvement through interactive visualizations and business intelligence techniques.

---

## 🎯 Project Objectives

- Track fundraising performance using key KPIs.
- Compare fundraising goals with actual amounts raised.
- Analyze donor engagement and marketing effectiveness.
- Identify top-performing fundraising methods and platforms.
- Support strategic planning using Root Cause Analysis (RCA) and SWOT Analysis.

---

## 📊 Dashboard 1: Overview Analysis

- KPI Cards
  - Total Funds
  - Total Fundraising Goal
  - Total Amount Raised
  - Total Marketing Spend
  - Social Media Reach
- Amount Raised by Organization
- Fundraising Platform Analysis
- Fund-wise Performance
- Interactive Slicers (Campaign Type & Fundraising Method)

---

## 📈 Dashboard 2: Methods, Strategies & Process Analysis

- Profit Analysis by Fundraising Method (Waterfall Chart)
- Campaign Type Distribution
- Monthly Amount Raised vs Marketing Spend
- Marketing ROI Analysis
- Fundraising Trend Analysis

---

## 🎯 Dashboard 3: Strategic Decision Making

- Fishbone (Ishikawa) Diagram
- SWOT Analysis
- Root Cause Analysis
- Strategic Recommendations for Process Improvement

---

## 💡 Key Insights

- Achieved **$7M** fundraising against a **$13M** target.
- Compared fundraising performance across organizations and platforms.
- Evaluated marketing spend versus fundraising returns.
- Identified the most effective fundraising methods and campaign types.
- Highlighted operational challenges and strategic improvement opportunities.
